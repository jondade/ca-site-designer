Style Guide

This is the style I use on my websites and pages. It includes typography, colors, buttons, and form elements.

How to use this Style Guide

There are two suggested ways to use this style guide.

  • Reference it when creating new web pages to ensure consistency in design.
  • Use the provided CSS classes and styles directly in your HTML code.
<link rel="stylesheet" href="https://jondade.github.io/ca-site-designer/styles.css">

Colours

This section outlines the color palette used in my designs.

  • Primary Color: #303030 (Dark Grey)
  • Accent Color: #d3d3d3 (Light Grey)
  • Background Color (light): #f4f4f4 (White)
  • Background Color (dark): #303030 (Dark Grey)
  • Text Color (light background): #303030 (Dark Grey)
  • Text Color (dark background): #f4f4f4 (White)

Code Blocks

Code blocks are styled with a dark background and light text for optimal readability. They should be 80 characters wide and minimum 5 lines high. They should be in the middle of the page, inset to show the code clearly.

    return "Hello, " + name + "!";

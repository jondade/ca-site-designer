Style Guide

This is the style I use on my websites and pages. It includes typography, colors, buttons, and form elements.

How to use this Style Guide

There are two suggested ways to use this style guide.

  • Reference it when creating new web pages to ensure consistency in design.
  • Use the provided CSS classes and styles directly in your HTML code.
<link rel="stylesheet" href="https://jondade.github.io/ca-site-designer/styles.css">
